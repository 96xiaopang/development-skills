# ThinkPHP 8.0 开发文档

这是 ThinkPHP 8.0 的本地离线开发文档。

## 简介
ThinkPHP 是一个免费开源的，快速、简单的面向对象的轻量级 PHP 开发框架。本目录包含了 ThinkPHP 8.0 的完整开发手册，涵盖了从基础安装、架构原理到各个功能模块（路由、控制器、模型、视图等）的详细说明。
